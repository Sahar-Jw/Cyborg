# CYBORG Gaming Platform

## Overview
CYBORG is a modern, responsive landing page for a gaming platform designed to showcase popular games, allow users to browse titles, manage their personal gaming library, and access streams/details. It features a dark cyberpunk theme perfect for esports and gaming communities.

The website welcomes users with a hero banner encouraging them to "BROWSE OUR POPULAR GAMES HERE", displays trending games with ratings and download stats, and shows a personalized library section tracking playtime and download status for games like Dota 2, Fortnite, and CS:GO.

## Key Features
- **Navigation**: Responsive navbar with Home, Browse, Details, Streams, and Profile links. Includes search bar and mobile offcanvas menu.
- **Hero Section**: Eye-catching banner with call-to-action to browse games.
- **Popular Games**: Grid of 8 trending games (e.g., Fortnite, PUBG, Dota 2, CS-GO) with ratings (4.8 stars), studios, and download counts (2.3M).
- **Gaming Library**: User's library showing added date, hours played, current status (Downloaded/Download), and action buttons.
- **Responsive Design**: Fully mobile-friendly with media queries for tablets and phones.
- **Interactive Elements**: Hover effects on buttons, profile, and links; Bootstrap-powered offcanvas menu.

## Technologies & Programs Used
- **HTML5**: Semantic structure for header, main sections (hero, popular, library), and footer.
- **CSS3**: Custom styles with CSS custom properties (variables), flexbox/grid layouts, responsive design via media queries. Uses SCSS-like nesting (compiled to CSS).
- **Bootstrap 5.3**: Responsive framework for navbar, offcanvas menu, buttons, and utilities.
- **Font Awesome 7**: Icons for search, stars, downloads, stars.
- **Assets**: High-quality gaming images (thumbnails, banners, avatars) organized in `assets/img/` and `assets/guide/`.

No JavaScript frameworks or backend; pure frontend static site.
